## What is it?

A Midone powered, laravel built admin dashboard, fully customized with an auth capabilities



<a name="usage"></a>
## Installation

Run:

```php
composer require olat-nji/midone-admin
```

Then:


```php
php artisan midone-admin:install
```

Then run migrations:

```php
php artisan migrate:fresh --seed
```

Now Enjoy:
